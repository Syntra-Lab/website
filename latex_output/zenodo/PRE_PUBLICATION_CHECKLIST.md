# Pre-Publication Checklist

## Before Uploading to Zenodo

### Content Review
- [ ] All content reviewed for accuracy
- [ ] No confidential/personal information
- [ ] No API keys, tokens, or secrets
- [ ] No draft notes or TODO comments
- [ ] No internal-only information

### Quality Check
- [ ] All LaTeX compiles without errors
- [ ] All PDFs render correctly
- [ ] Math equations display properly
- [ ] Arabic text renders correctly
- [ ] Images/figures are included

### Metadata
- [ ] Title is accurate and descriptive
- [ ] Authors/contributors listed
- [ ] Keywords/tags added
- [ ] Description written
- [ ] License specified (CC-BY-4.0 recommended)

### Post-Publication
- [ ] DOI recorded
- [ ] Citation generated
- [ ] Linked from GitHub README
- [ ] Syntra community joined

---

## Workflow

```
GitHub (living) → Review → Zenodo (permanent)
     ↑                ↓
   Edit          New Version
     ↑                ↓
   Drafts      (after review)
```

**Important:** Zenodo versions cannot be deleted. Only publish content that has been reviewed and is finalized.
